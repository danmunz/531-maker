# Rules for Assigning Accessory Lifts

If the workout includes Squat or Deadlift, then add either core work or single-leg/hamstring work.
If the workout includes Deadlift, then avoid extra heavy posterior-chain or lower-back loading.
If the workout includes Squat, then avoid adding a lot of extra bilateral leg fatigue.
If the workout includes Bench Press or Overhead Press, then add pulling work.
If the workout includes a lot of pressing, then make pull volume at least equal to push volume.
For normal training weeks, keep assistance simple: 1 pull; 1 push or arms; and 1 core or single-leg/hamstring movement.
If Bench Press is the primary 5/3/1 lift, then make the push-or-arms slot a chest push by default.
If Bench Press is only a secondary FSL lift on a lower-body-heavy day, then the push-or-arms slot may be a low-fatigue chest push, but only after pulling and recovery needs are covered.
If the workout includes Overhead Press, then consider extra upper-back or rear-delt work.
If a main lift already heavily trains a muscle group, then do not pile on another heavy barbell accessory for that same area.
If an accessory can support the main lift with less fatigue, then prefer that option.
If a movement pattern is missing from the workout, then use accessories to fill the gap.
If the workout is lower-body heavy, then make the third slot core or a lighter single-leg/hamstring movement.
If the workout is upper-body heavy and Overhead Press-focused, then make the push-or-arms slot triceps or shoulder-balance work.
Across a four-week block, repeat the same accessory slot pattern for each day type, but rotate only a small pool of exercises in each slot so progression stays easy to track.
In deload week, reduce assistance to 2 movements total: 1 pull and 1 core or light lower-body movement, for 2 to 3 easy sets each.
If choosing between two accessory options, then pick the one that helps the main lifts; improves push/pull balance; adds muscle; and creates less recovery cost.
